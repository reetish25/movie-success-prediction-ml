# Movie Success Prediction using Machine Learning

## Overview
This project predicts movie success using machine learning techniques based on features such as budget, popularity, runtime, vote count, ratings, and revenue.

## Dataset
- Movie metadata dataset
- ~4800 movie records

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Models Implemented
- Logistic Regression
- Decision Tree
- Random Forest

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
